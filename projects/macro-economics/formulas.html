📐第一部分：全章节核心公式速记

第2章 GDP核算

Y = C + I + G + NX
GDP平减指数 = 名义GDP/实际GDP × 100
CPI = (固定篮子当期成本/基期成本) × 100
失业率 = 失业人数/劳动力 × 100%
GDP只计最终产品、当期生产、境内。二手货、股票、转移支付不计入。CPI含进口品，GDP平减指数不含。

第3章 古典模型（长期）

Y = F(K, L) = A·Kα·L1-α
W/P = MPL（实际工资 = 劳动边际产出）
R/P = MPK（实际租金 = 资本边际产出）
S = I（可贷资金市场均衡）
S = (Y-T-C) + (T-G) = Sprivate + Spublic
古典二分法：实际变量由供给侧决定，货币只影响名义变量。价格完全弹性。

第4章 货币系统

M = m × B，m = (cr+1)/(cr+rr)
B = C + R（基础货币 = 通货 + 准备金）
M1 = C + D（通货 + 活期存款）
央行控制B，但m取决于公众(cr)和银行(rr)行为。R↑→rr↑→m↓，M可能不升反降。

第5章 通货膨胀

MV = PY（数量方程式）
π = ΔM/M + ΔV/V - ΔY/Y
i = r + π（费雪方程）
货币中性：长期中M只影响P和i，不影响Y和r。费雪效应：Δπ→Δi一对一传导。

第6章 索洛增长模型

Δk̂ = sf(k̂) - (δ+n+g)k̂（资本积累方程）
稳态：sf(k̂*) = (δ+n+g)k̂*
k̂* = [s/(δ+n+g)]1/(1-α)（Cobb-Douglas）
黄金律：MPK = δ + n + g
MPK = α·k̂α-1 = α/(K/Y)
Y增长率 = n+g；y增长率 = g；ŷ增长率 = 0
增长核算：ΔY/Y = ΔA/A + α·ΔK/K + (1-α)·ΔL/L
核心结论：s只有水平效应没有增长效应。长期增长率仅由g决定。

第7章 经济波动导论

奥肯定律：Δ(Y-Ȳ)/Ȳ ≈ -2Δu
失业↑1%→GDP↓约2%。短期P粘性→货币非中性；长期P弹性→货币中性。

第8章 IS-LM模型（短期）

IS：Y = C(Y-T) + I(r) + G
LM：M/P = L(Y, r)
乘数 = 1/(1-MPC)（凯恩斯交叉）
税收乘数 = -MPC/(1-MPC)
平衡预算乘数 = 1
政策效果：G↑/T↓→IS右移→Y↑+r↑。挤出效应：r↑→I↓→部分抵消Y的初始增加。

第9-10章 总供给与开放经济

SRAS：Y = Ȳ + α(P - Pe)
菲利普斯：π = πe - γ(u - un) + ε
牺牲率 ≈ 降1%通胀需牺牲的GDP百分比
NX = S - I（贸易余额 = 国民储蓄 - 国内投资）
小型开放经济：r = r*（国内利率 = 世界利率）

第11-12章 蒙代尔-弗莱明模型

IS*：Y = C(Y-T) + I(r*) + G + NX(e)
LM*：M/P = L(r*, Y)（r被r*锁定，LM垂直）
浮财无货有，固财有货无。不可能三角：独立货币、资本自由、固定汇率三选二。

✅第二部分：判断题高频考点与改错模板

⚠️ 判断题答题格式：先写"正确"或"错误"。如果错误，必须写出改正后的正确表述。每题2分，判断+改正各1分。

🔴 GDP与福利

"GDP是衡量经济福利的完美指标。"
改：GDP遗漏家庭生产、闲暇、环境质量、收入分配、地下经济，不是完美福利指标。
"买股票属于GDP中的投资。"
改：买股票是金融资产转移，不计入GDP。GDP中的I是实物资本形成。
"政府发放的养老金计入GDP中的G。"
改：养老金是转移支付，不计入GDP。G只含政府对商品和服务的购买。

🔴 古典模型

"古典模型中，价格完全弹性，货币供给变化只影响名义变量。"
这就是古典二分法/货币中性。正确表述，无需改正。
"古典模型中，政府增加购买支出可以提高均衡产出。"
改：古典模型中Y由K、L和技术决定（供给侧），G↑不改变Y，只抬高利率挤出投资。

🔴 索洛增长模型（高频！）

"储蓄率提高可以永久提高人均产出增长率。"
改：s↑只有水平效应（y*提高），没有增长效应。长期增长率仍由技术进步率g决定。
"两个国家有相同的人口增长率，就会收敛到相同的人均产出。"
改：需要所有结构参数相同（s、δ、n、g、生产函数）才会收敛——这是条件收敛，不是绝对收敛。
"如果MPK > δ+n+g，说明资本存量高于黄金律水平。"
改：MPK > δ+n+g→资本边际产出高于持平投资斜率→资本不足→k* < k_gold→应增加储蓄。
"人口更多的国家，稳态人均产出更高。"
改：人口规模L不影响人均稳态值。y*由s、δ、n、g决定，与L大小无关（规模报酬不变）。

🔴 IS-LM模型

"流动性陷阱中，货币政策最有效。"
改：流动性陷阱中LM水平→货币增发无法再降低r→货币政策无效，财政政策完全有效。
"IS-LM中财政扩张的乘数等于凯恩斯交叉中的乘数。"
改：IS-LM中G↑→r↑→I↓（挤出效应）→乘数小于凯恩斯交叉（凯恩斯交叉假设r不变）。
"LM曲线垂直时（古典情形），财政政策最有效。"
改：LM垂直→货币需求与利率无关→G↑只会抬高r，完全挤出I→财政政策完全无效。

🔴 菲利普斯曲线

"政策制定者可以永久用较高通胀换取较低失业。"
改：长期菲利普斯曲线在自然失业率处垂直→不存在长期通胀-失业权衡。只有短期有。
"理性预期下，反通胀的牺牲率为正。"
改：理性预期+可信承诺下，公众立即调整预期→牺牲率可接近零。适应性预期下牺牲率为正。
"根据卢卡斯批判，历史数据估计的模型参数在政策变化后依然有效。"
改：卢卡斯批判：政策规则改变→人们行为（预期）改变→历史参数不再有效。

🔴 开放经济与M-F模型

"浮动汇率下，财政扩张能有效提高产出。"
改：浮动汇率下G↑→本币升值→NX完全抵消→Y不变。浮动汇率下货币政策有效，财政无效。
"固定汇率下，货币扩张能有效提高产出。"
改：固定汇率下M↑→资本外流→央行干预→M自动回缩→Y不变。固定汇率下财政政策有效，货币无效。
"一国可以同时实现独立货币政策、资本自由流动和固定汇率。"
改：不可能三角——三者只能选其二。中国选独立货币+管理汇率→必须限制资本流动。

🔴 费雪效应与通胀

"货币供给增长率提高会永久降低实际利率。"
改：实际利率r由实体经济（S=I）决定，不受货币影响（货币中性）。货币增长↑只提高名义利率（费雪效应）。
"通货膨胀对所有人都造成损失。"
改：未预期通胀使债务人受益（还的钱实际价值降低）、债权人受损。预期到的通胀影响较小。

📝第三部分：计算分析题答题模板 — 索洛模型（20分）

📐 题型A：求稳态 + 黄金律判断 + 储蓄率调整

Step 1写出稳态条件：sf(k̂*) = (δ+n+g)k̂*
Step 2Cobb-Douglas下代入：k̂* = [s/(δ+n+g)]1/(1-α)，计算数值
Step 3计算增长率：ŷ→0%，y→g，Y→n+g（三个增长率对应三个层次）
Step 4计算MPK：MPK = α·(k̂*)α-1 = α/(K/Y)
Step 5黄金律判断：比较MPK vs δ+n+g。MPK大→资本不足→增s；MPK小→资本过度→降s
Step 6黄金律k̂_gold：MPK=δ+n+g → k̂_gold = [α/(δ+n+g)]1/(1-α)
Step 7黄金律s_gold：s_gold = (δ+n+g)·k̂_gold/f(k̂_gold)
📌 必写结论句：该经济体MPK=XX% > (=/<) δ+n+g=XX%，资本存量低于（等于/高于）黄金律水平，应增加（不变/减少）储蓄率。过渡期间人均产出增长率暂时高于（等于/低于）稳态增长率g。

📐 题型B：增长核算

Step 1计算各变量的增长率：ΔY/Y, ΔK/K, ΔL/L
Step 2代入：ΔA/A = ΔY/Y - α·ΔK/K - (1-α)·ΔL/L
Step 3分别写：资本贡献=α·ΔK/K=XX%，劳动贡献=(1-α)·ΔL/L=XX%，TFP贡献=XX%
📌 结论句：经济增长的XX%可由要素投入解释，剩余XX%归因于全要素生产率（索洛残差）。

📝第四部分：计算分析题答题模板 — IS-LM模型（25分）

📐 标准IS-LM六步求解法

Step 1推导IS曲线：Y = C + I + G，代入消费函数C=a+b(Y-T)和投资函数I=c-dr，整理为 Y = A - B·r 形式
Step 2推导LM曲线：M/P = kY - hr，整理为 r = (k/h)Y - (1/h)(M/P) 或 Y = (1/k)(M/P) + (h/k)r
Step 3联立求均衡：IS与LM联立 → 解出 Y* 和 r*
Step 4政策冲击分析：
财政扩张(G↑或T↓)→IS右移→新IS方程→联立新LM→新的Y**和r**
货币扩张(M↑)→LM右移→新LM方程→联立IS→新的Y***和r***
Step 5挤出效应计算：挤出 = 凯恩斯乘数×ΔG - 实际ΔY
凯恩斯乘数 = 1/(1-MPC)，ΔY_凯恩斯 = 乘数×ΔG
实际ΔY从Step 4得出，两者之差即为利率上升挤出的部分
Step 6推导AD曲线（如需）：从IS和LM消去r，得到 Y = Y(P) 形式
📌 必写结论句：财政扩张使Y从XX增至XX，r从XX%升至XX%，挤出效应为XX。货币扩张使Y从XX增至XX，r从XX%降至XX%。两种政策对产出有相似效果但对利率和投资的影响方向相反。

📝第五部分：计算分析题答题模板 — 蒙代尔-弗莱明模型（20分）

📐 M-F模型标准求解流程

Step 1写出IS*方程：Y = C(Y-T) + I(r*) + G + NX(e)，代入具体函数
Step 2写出LM*方程：M/P = L(r*, Y)。注意r被世界利率r*锁定，LM*是垂直的（Y由货币市场单独决定）
Step 3浮动汇率下求解：
① 从LM*解出Y*（Y由货币市场决定）
② 将Y*代入IS*解出均衡汇率e*
③ 代入NX(e)得到贸易余额
Step 4固定汇率下求解：
① 将给定e代入IS*解出Y*
② 将Y*代入LM*解出所需的M（央行被动调整）
Step 5政策分析对比：
政策浮动汇率固定汇率
财政扩张G↑Y不变，ε↑，NX↓Y↑，M被动↑
货币扩张M↑Y↑，ε↓，NX↑Y不变，储备↓
贸易保护Y不变，ε↑，NX不变Y↑，NX↑
📌 必写结论句：浮动汇率下，货币政策有效而财政政策无效（汇率调整抵消财政效果）。固定汇率下，财政政策有效而货币政策无效（央行需维持汇率牺牲货币自主权）。这体现了"不可能三角"——独立货币政策、资本自由流动、固定汇率三者只能选其二。

🧠第六部分：易混淆概念辨析（判断改错高频区）

1. 水平效应 vs 增长效应

"储蓄率提高→增长率永久提高"
改：s提高→水平效应（y*提高），过渡期增长率暂时提高，但长期增长率回归g。只有g的变化才有增长效应。

2. 绝对收敛 vs 条件收敛

"穷国必然增长更快，最终追上富国"
改：只有结构参数（s,δ,n,g）相同时才收敛——这是条件收敛。无条件/绝对收敛的实证支持较弱。

3. 短期 vs 长期

"货币供给减少使产出永久下降"
改：短期P粘性→Y↓；长期P弹性→Y回Ȳ，只P同比例↓。货币长期中性。

4. 名义利率 vs 实际利率

"货币扩张降低名义利率"
改：短期M↑→LM右移→r↓→i↓（流动性效应）。长期M↑→π↑→费雪效应→i↑。短期和长期方向相反！

5. CPI vs GDP平减指数

"CPI和GDP平减指数衡量的是同样的价格变化"
改：CPI衡量消费者购买的固定篮子（含进口品），GDP平减指数衡量国内生产的所有商品（不含进口品）。进口石油涨价→CPI涨更多。

6. 适应性预期 vs 理性预期

"无论哪种预期，反通胀都有显著的产出代价"
改：适应性预期下反通胀有牺牲率（约5%GDP/1%通胀）。理性预期+可信承诺下牺牲率可接近零。

7. 索洛模型 vs 内生增长模型

"索洛和AK模型都预测储蓄率不影响长期增长率"
改：索洛：s不影响长期增长率（仅水平效应）。AK模型：s永久影响增长率（sA-δ-n），因为消除了边际报酬递减。
