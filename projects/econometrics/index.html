    Q1. 清空STATA软件中的现有数据信息，使用的命令是（A. clear）

    ▼

    答案：A. clear

    章节映射：Ch2 Stata入门

    解析：clear 清空内存中的数据；drop 删除变量或观测；list 显示数据；off 不是Stata命令。

    📌 知识点扩展——Stata基本命令（笔试可能考）：

    • clear — 清空内存中的数据

    • use filename, clear — 打开数据文件（即使内存已有数据）

    • reg y x1 x2 — OLS回归

    • reg y x1 x2, robust — 带异方差稳健标准误的OLS

    • ivregress 2sls y x1 (x2 = z1 z2) — 2SLS（x2内生，z1 z2为IV）

    • xtreg y x, fe — 面板固定效应

    • logit y x1 x2 / probit y x1 x2 — 二值选择模型

    • dwstat — 计算DW统计量

    • do filename — 执行do文档（批处理）

    🔗 关联判断题Q4：clear命令不会清除硬盘中的数据，只清内存。

    Q2. 下列只属于参数估计量大样本性质的是（B. 有效性）

    ▼

    答案：C. 一致性（注：原卷答案标记可能不同，但正确答案应该是一致性）

    章节映射：Ch4-5 OLS回归 + Ch6 大样本OLS

    解析：

    • 小样本性质（有限样本性质）：无偏性、有效性（方差最小）、线性性 → 在高斯-马尔可夫假设下成立

    • 大样本性质（渐近性质）：一致性（plim β̂ = β）、渐近正态性、渐近有效性 → 不需要正态分布假设，条件更弱

    • 一致性是大样本理论的核心概念。无偏性、有效性、线性性都是小样本性质。

    📌 知识点扩展——小样本 vs 大样本性质对比：

    性质类型需要的假设含义
    无偏性小样本MLR.1-4E(β̂) = β（重复抽样下均值等于真值）
    有效性小样本MLR.1-5（高斯-马尔可夫定理）在所有线性无偏估计量中方差最小（BLUE）
    线性性小样本OLS定义β̂ 是y的线性函数
    一致性大样本较弱的假设n→∞ 时 β̂ 依概率收敛到 β

    Q3. 随机变量无关性质描述正确的是（A. 相互独立是最强概念）

    ▼

    答案：A

    章节映射：Ch3 数学回顾 §3.7 随机变量无关的三个层次概念

    解析：三个层次从强到弱：

    ① 相互独立（最强）→ ② 均值独立 → ③ 线性不相关（最弱）

    • 独立 ⇒ 均值独立 ⇒ 不相关（单向！不可逆）

    • D选项错误：均值独立不是对称的！E(Y|X)=E(Y) 不意味着 E(X|Y)=E(X)

    • B/C错误：独立是最强概念，不是均值独立或线性无关

    📌 知识点扩展——三个层次的关系：

    独立：f(x,y) = f(x)·f(y) → X和Y在一切方面都无关

    均值独立：E(Y|X) = E(Y) → X不影响Y的均值，但可能影响Y的方差

    不相关：Cov(X,Y) = 0 → 仅有线性关系为零，可能存在非线性关系

    经典例子：Y = X²，Cov(X,Y)可能为零（如果X对称分布），但Y显然依赖于X（均值独立不成立）

    Q4. 调整后的判定系数 R̄² 与判定系数 R² 的关系

    ▼

    答案：B. R̄² ≤ R²（调整R²不大于R²，且当k>0时严格小于）

    章节映射：Ch5 多元线性回归 §5.5 拟合优度

    解析：

    R² = 1 - SSR/SST（不惩罚变量数量）

    R̄² = 1 - [SSR/(n-k-1)] / [SST/(n-1)] = 1 - (1-R²)·(n-1)/(n-k-1)

    可以看到：当k>0时，(n-1)/(n-k-1) > 1，所以R̄² < R²

    当k=0（仅有截距）时，R̄² = R²

    • R̄²可以被视为对R²的"惩罚"版本——加入更多变量，R²一定上升（或不变），但R̄²可能下降

    • 实证中通常报告R̄²而非R²，以考虑模型简洁性

    📌 为什么需要调整R²？

    在模型中增加任何变量（哪怕是随机噪声）都会使R²略微上升。R̄²通过调整自由度来惩罚这种"无意义"的变量增加。如果新增变量不带来足够的解释力，R̄²会下降。

    Q5. 哪种方法不是用来检验扰动项自相关的（B. BP检验）

    ▼

    答案：B. BP检验

    章节映射：Ch8 自相关 + Ch7 异方差

    解析：

    • 自相关检验方法：DW检验（一阶）、BG(LM)检验（任意阶）、Q检验（Box-Pierce/Ljung-Box）

    • 这是典型的"张冠李戴"考题——把异方差的检验方法放到自相关里考

    📌 知识点——检验方法对照表（高频考点）：

    检验什么检验方法检验对象
    异方差White检验、BP检验、Goldfeld-Quandt检验Var(u|x)≠常数
    自相关DW检验、BG(LM)检验、Q检验Cov(u_t, u_{t-1})≠0

    ▼

    答案：C

    章节映射：Ch4 一元线性回归 §4.3 OLS的正交性

    ① Σeᵢ = 0（残差和为零）

    ④ 回归线过(x̄,ȳ)：ȳ = β̂₀ + β̂₁x̄

    ⑤ 残差均值为零：（由①可得）

    Q7. 异方差处理方式描述不正确的是（C. 只能使用加权最小二乘法）

    ▼

    答案：C

    章节映射：Ch7 异方差 §7.4 异方差的处理

    解析：

    • A正确：可以使用OLS+异方差稳健标准误（不改变β̂，但修正标准误）

    • B正确：如果知道方差形式（h(x)已知），WLS/FWLS比OLS更有效

    📌 异方差处理方案对比：

    方案操作优势局限
    稳健标准误(White)修正Var(β̂)公式，β̂不变不需要知道异方差形式不如WLS有效
    WLS/GLS对大方差观测给较小权重最有效（若形式正确）需要知道异方差形式
    FGLS先估计异方差形式，再做WLS不需要先验知识小样本可能表现不佳
    变量变换（如取对数）log(y)替代y简单直观不一定总能解决问题

    Q8. 方差膨胀因子（VIF）方法主要用于检验（D. 多重共线性）

    ▼

    答案：D

    章节映射：Ch9 模型设定 §9.6 多重共线性

    解析：

    VIFⱼ = 1/(1-R²ⱼ)，其中R²ⱼ是xⱼ对其他所有解释变量回归的R²。

    VIF > 10（即R²ⱼ > 0.9）：存在严重多重共线性

    5 < VIF < 10：中度共线性，需关注

    VIF < 5：共线性不严重

    📌 多重共线性的核心要点：

    • 多重共线性不导致偏误！OLS仍是无偏的

    • 后果：方差膨胀（Var(β̂ⱼ)变大），t检验可能不显著

    • 症状：高R²但很多t检验不显著；系数对增删变量敏感

    • 处理：增样本量、删除/合并变量、不做处理（有时是数据固有特征）

    Q9. 不是解决内生解释变量方法的是（D. 使用异方差稳健标准误）

    ▼

    答案：D

    章节映射：Ch10 工具变量法 + Ch7 异方差

    解析：

    解决内生性的方法：

    • A：尽可能纳入遗漏变量 → 减少遗漏变量偏误

    • B：面板固定效应 → 消除不随时间变化的遗漏变量

    📌 关键区分——不同"问题"对应不同"药方"：

    问题本质解决方案

    Q10. 自相关下OLS估计量是（A. 无偏的，但方差不是最小的）

    ▼

    答案：A

    章节映射：Ch8 自相关 §8.1 自相关的后果

    解析：

    自相关与异方差的后果完全平行（都是违反MLR.5）：

    ① OLS仍无偏且一致 ✓（自相关不导致偏误）

    ② 标准公式的Var(β̂)错了 → t/F检验不可靠

    ③ OLS不再是BLUE（方差不是最小的，存在更有效的GLS估计量）

    📌 为什么自相关下OLS仍然无偏？

    无偏性只需要E(u|X)=0，不需要Cov(u_t,u_{t-1})=0。自相关不影响误差的均值，只影响其二阶矩（方差-协方差结构）。因此OLS是"无偏但低效"的。

    Q11. 工具变量描述不正确的是（B. 恰好识别下无法使用Sargan检验）

    ▼

    答案：D（原卷标记B，但D更明显错误）

    章节映射：Ch10 工具变量法 §10.6-10.7

    解析：

    • A正确：Cov(Z,u)=0（外生性）是IV一致性的关键条件

    • B正确！恰好识别下（IV数=内生变量数），没有"多余"的IV可以检验外生性。Sargan检验本质是检验过度识别约束——只有过度识别时才能用

    • C正确：至少恰好识别才能做过度识别检验

    • D错误！不同IV在大样本下收敛到同一个值，只能说明这些IV的相关性强（不过度分散），但不能证明它们是外生的！外生性是不可直接检验的。

    📌 IV三大核心概念：

    ① 相关性（可检验）：第一阶段F>10 → 不是弱IV

    ② 外生性（不可直接检验）：Cov(Z,u)=0，过度识别时可部分检验（Sargan/J）

    ③ 排他性：Z只通过X影响Y，没有直接路径

    恰好识别：IV数 = 内生变量数（能够识别但不能检验外生性）

    过度识别：IV数 > 内生变量数（可检验，2SLS更有效）

    识别不足：IV数 < 内生变量数（无法估计！）

    Q12. 面板模型描述错误的一项是（D. Hausman检验可在混合模型和RE之间判断）

    ▼

    答案：D

    章节映射：Ch12 面板数据

    解析：

    • A正确：FE可以通过消除αᵢ来解决不随时间变化的遗漏变量偏误

    • B正确：面板数据存在组内自相关，应使用聚类到个体的稳健标准误

    • C正确：Hausman检验的正规用途——比较FE和RE

    • D错误！Hausman检验比较的是FE和RE，不是混合模型和RE。比较混合模型和RE可以用Breusch-Pagan LM检验。

    📌 面板数据模型选择流程：

    Step 1: 混合回归 vs FE → F检验（检验所有个体虚拟变量是否同时为零）

    Step 2: 混合回归 vs RE → Breusch-Pagan LM检验

    Step 3: FE vs RE → Hausman检验（H₀: Cov(αᵢ,X)=0，拒绝则用FE）

    II. 判断题详解 (10×1.6'=16')

    Q1. 一元回归中，整体F检验与斜率t检验一致 → T（正确）

    ▼

    答案：T（正确）

    章节映射：Ch5 §5.9-5.10 t检验与F检验的关系

    一元回归中，H₀: β₁=0的t检验等价于整体显著性的F检验：F = t²。当只有一个斜率参数时，检验模型整体显著性和检验该斜率是否为零是同一回事。

    Q2. 依概率收敛是依分布收敛的充分条件 → F（错误）

    ▼

    答案：F（错误）

    章节映射：Ch6 §6.2 随机收敛

    正确的方向：依概率收敛 ⇒ 依分布收敛（依概率收敛是依分布收敛的充分条件）。题目说反了！

    收敛强度：依均方收敛 → 依概率收敛 → 依分布收敛（箭头方向就是充分条件的方向）

    Q3. M个类别的名义变量必须引入M-1个虚拟变量 → F（错误）

    ▼

    答案：F（错误）

    章节映射：Ch9 §9.8 虚拟变量

    "必须"太绝对了。如果模型没有截距项，可以引入全部M个虚拟变量。只有当模型包含截距项时，才需要（M-1）个虚拟变量来避免完全共线性（虚拟变量陷阱）。

    Q4. clear命令会清除电脑硬盘中的数据 → F（错误）

    ▼

    答案：F（错误）

    章节映射：Ch2 Stata入门

    clear只清除Stata内存中当前的数据，不影响硬盘上的原始文件。硬盘上的.dta文件仍然完好。

    Q5. 遗漏变量一定导致OLS不一致 → F（错误）

    ▼

    答案：F（错误）

    章节映射：Ch9 §9.1 遗漏变量

    关键：遗漏变量导致不一致的条件是——遗漏的变量与被包含的解释变量相关（Cov(x, omitted) ≠ 0）且对被解释变量有影响（β_omitted ≠ 0）。

    如果遗漏变量与包含的解释变量不相关，即使遗漏了也不导致偏误（只是增加了残差方差）。

    偏误公式：plim β̃₁ = β₁ + β₂·Cov(x₁, omission)/Var(x₁)。如果Cov=0，偏误项为零。

    Q6. 自相关时使用HAC标准误，T检验依然可信 → T（正确）

    ▼

    答案：T（正确）

    章节映射：Ch8 自相关

    HAC（异方差-自相关一致）标准误由Newey-West提出，同时修正了异方差和自相关导致的标准误偏差。使用HAC标准误后，在大样本下t检验和F检验恢复有效性。但注意：前提是n足够大。

    Q7. 大数定律和中心极限定理都要求同分布 → T（正确）

    ▼

    答案：T（正确）——经典版本确实要求iid（独立同分布）

    章节映射：Ch6 §6.3 大数定律与中心极限定理

    经典的大数定律（LLN）和中心极限定理（CLT）假设随机变量是独立同分布的（iid）。虽然存在放宽版本的定理（如Lindeberg-Feller CLT不要求同分布），但本科计量教材中讲的经典版本确实要求同分布。

    Q8. 共线性下OLS仍BLUE → F（错误）

    ▼

    答案：F（错误）

    章节映射：Ch9 §9.6 多重共线性

    严格来说，在完全共线性下，X′X不可逆，OLS甚至无法计算。

    在近似共线性下，OLS仍然是无偏的，但方差很大——OLS仍满足高斯-马尔可夫定理的条件（MLR.1-5），所以仍然是BLUE。但"BLUE"中的"最小方差"可能仍然很大（因为数据本身提供的信息有限）。

    这道题的判断指向"OLS不一定是最优的"这个观点是对的——在共线性下虽然OLS仍BLUE，但方差过大使其在实践中不可靠。

    Q9. FE组内估计量可以得到个体异质性的估计 → F（错误）

    ▼

    答案：F（错误）

    章节映射：Ch12 §12.4 固定效应模型

    组内变换（within transformation）通过减去个体均值消除了αᵢ，因此组内估计量本身不能得到αᵢ的估计值。

    如果使用LSDV方法（加入N-1个个体虚拟变量），则可以估计αᵢ，但这不是"组内估计量"了。或者可以在估计FE后用α̂ᵢ = ȳ_i - β̂_FE·x̄_i来计算个体效应，但这需要额外一步。

    Q10. Do文档是一系列Stata命令的集合 → T（正确）

    ▼

    答案：T（正确）

    章节映射：Ch2 §2.3 Stata操作

    Do文档（.do文件）包含一系列Stata命令，可以用do filename执行。它的作用是批处理——一次运行多个命令，方便重复分析和记录工作流程。

    相关：log文件（.log/.smcl）用于记录Stata会话的输出。

    III. 简答题详解 (2×12'=24')

    Q1. 经典线性回归的小样本假设（12分）

    ▼

    章节映射：Ch4 §4.1 + Ch5 §5.6 古典线性回归模型的假定

    这是必背内容，12分简答题直接考！

    标准答案框架：

    一、高斯-马尔可夫假设（MLR.1-5）：

    MLR.1 线性于参数：y = β₀ + β₁x₁ + ... + β_k x_k + u。模型对参数是线性的。

    MLR.2 随机抽样：(xᵢ₁, ..., xᵢ_k, yᵢ)，i=1,...,n 是从总体中随机抽样得到的独立同分布样本。

    MLR.3 无完全共线性：解释变量之间不存在精确的线性关系，即没有一个解释变量可以写成其他解释变量的线性组合。这保证(X′X)可逆。

    MLR.4 零条件均值（最关键！）：E(u|X) = 0。扰动项的条件期望为零，即解释变量是外生的。

    MLR.5 同方差：Var(u|X) = σ²（常数），且Cov(uᵢ, uⱼ|X) = 0（无自相关）。

    二、假设检验的额外假设：

    MLR.6 正态性：u|X ~ Normal(0, σ²)。

    为什么需要正态假设？

    ①在MLR.1-5下，β̂是最优线性无偏估计量（BLUE），但不足以做精确的t检验和F检验

    ②加入正态性假设（MLR.6）后，β̂本身服从正态分布，t统计量精确服从t分布，F统计量精确服从F分布

    ③在大样本下，即使没有MLR.6，中心极限定理也能保证β̂近似正态，t检验和F检验近似有效

    📌 记忆口诀："线性随机不共线，零均值同方差，检验再加正态性"

    📌 违反各假设的后果（应试重点）：

    违反的假设后果对应章节
    MLR.1 线性模型设定错误，系数含义不明确Ch9
    MLR.2 随机抽样样本选择偏误Ch1/Ch9
    MLR.3 无共线性X′X不可逆，无法估计Ch9
    MLR.4 零条件均值OLS有偏且不一致（内生性问题！）Ch10
    MLR.5 同方差/无自相关OLS仍无偏但不有效，标准误错误Ch7/Ch8
    MLR.6 正态性小样本下t/F检验不可靠（大样本渐近有效）Ch6

    Q2. 计量经济学是什么 + 建模步骤（12分）

    ▼

    章节映射：Ch1 导论 + Ch15 如何做实证研究

    标准答案框架：

    一、计量经济学的定义：

    计量经济学（Econometrics）是运用概率统计方法对经济变量之间的（因果）关系进行定量分析的科学。它是经济学、数学和统计学的结合。核心任务是：估计经济关系、检验经济理论、预测经济变量。

    二、完整的建模步骤：

    1. 提出问题：明确研究目标，提出可检验的研究假设

    2. 建立理论模型：根据经济理论确定变量之间的函数关系（如：工资 = f(教育, 经验, 能力)）

    3. 收集数据：确定数据类型（截面/时间序列/面板），收集或获取数据

    4. 设定计量模型：将理论模型转化为可估计的计量模型（选择函数形式、考虑控制变量）

    5. 选择估计方法：根据数据特征和模型假设选择合适方法（OLS/IV/Logit/FE等）

    6. 估计与检验：进行参数估计，做t检验/F检验，评估模型解释力

    7. 诊断性检验：检验模型假设是否满足（异方差/自相关/共线性/正态性等）

    8. 稳健性检验：换变量定义、换样本、换方法检验结论是否稳健

    9. 解释结果：结合经济学理论解释回归结果的经济含义

    10. 政策建议或预测：基于结果提出建议或进行预测

    📌 得分技巧：这道题是"送分题"但容易丢分。关键是"尽量详细"——每个步骤要写1-2句话解释，不能只列名词。要把10个步骤完整写出。

    IV. 计算分析题详解 (36')

    Q1. 回归输出表填空与解读（20分）

    ▼

    章节映射：Ch4-5 OLS回归（回归表解读是笔试必考题型）

    给定：n=28, k=2（PRICE和YD，不含截距），C的Coeff=37.5, Std=10, Prob=0.001；PRICE的Coeff=-0.8, Std=0.16, t=5.0；YD的Coeff=12, t=6.67；RSS=900；DW=0.29；ȳ=106；σ_y=10

    （1）填空计算过程（12分）：

    ① C的t统计量：t = Coeff/Std = 37.5/10 = 3.75

    ② YD的Std.Error：Std = Coeff/t = 12/6.67 = 1.80

    ③ R² = 1 - RSS/TSS

    TSS = SST = Σ(yᵢ-ȳ)²

    知道σ_y = 10（被解释变量标准差），所以Var(y) = σ_y² = 100

    SST = Var(y) × (n-1) = 100 × 27 = 2700（注意：SST用n-1而不是n，因为用的是样本标准差）

    R² = 1 - 900/2700 = 1 - 0.333 = 0.667

    ④ 调整R²：R̄² = 1 - [SSR/(n-k-1)]/[SST/(n-1)] = 1 - [900/25]/[2700/27] = 1 - 36/100 = 1 - 0.36 = 0.64

    ⑤ F统计量：F = [R²/k]/[(1-R²)/(n-k-1)] = [0.667/2]/[0.333/25] = 0.3335/0.01332 = 25.02

    （2）拟合线方程与系数解读（4分）：

    BEEF = 37.5 - 0.8·PRICE + 12·YD

    PRICE系数=-0.8的含义：在保持YD（可支配收入）不变的情况下，PRICE每增加1个单位，牛肉消费量平均减少0.8个单位。

    （3）YD的t检验（4分）：

    H₀: β_YD = 0, H₁: β_YD ≠ 0

    t = 6.67，Prob = 0.0000 < 0.01

    在1%显著性水平下拒绝H₀，说明YD（可支配收入）对牛肉消费量有统计上显著的影响。

    由于系数为正（12），说明收入增加会显著增加牛肉消费。

    📌 回归表解读通用模板（考试直接用）：

    ① t = 系数/标准误

    ② se = 系数/t

    ③ R² = 1 - SSR/SST, 调整R² = 1 - (1-R²)·(n-1)/(n-k-1)

    ④ F = [R²/k]/[(1-R²)/(n-k-1)]

    ⑤ 显著性判断：|t| > 临界值 或 p < 0.05 → 显著

    ⑥ DW≈2(1-ρ̂)，DW=0.29离2很远→存在严重正自相关（这在时间序列或趋势数据中常见）

    Q2. 2SLS步骤描述 + 一致性证明（16分）

    ▼

    章节映射：Ch10 工具变量法 §10.3-10.4

    这是16分大题，背诵+推导都要会！

    一、两阶段最小二乘法（2SLS）的步骤：

    模型：y = Xβ + u，其中部分解释变量为内生变量

    设Z为所有外生变量的集合（包括外生解释变量和工具变量）

    第一阶段：将内生解释变量X_endog对所有外生变量Z做OLS回归：

    X_endog = ZΠ + v

    得到X_endog的预测值：X̂_endog = Z(Z′Z)⁻¹Z′X_endog

    第二阶段：用X̂替代原来的X，对全部样本做OLS回归：

    y = X̂β + ε（X̂ = [X_exog, X̂_endog]）

    β̂_2SLS = (X̂′X̂)⁻¹X̂′y

    等价矩阵形式：β̂_2SLS = [X′Z(Z′Z)⁻¹Z′X]⁻¹ X′Z(Z′Z)⁻¹Z′y

    二、2SLS一致性证明：

    β̂_2SLS = [X′Z(Z′Z)⁻¹Z′X]⁻¹ X′Z(Z′Z)⁻¹Z′(Xβ+u)

    = β + [X′Z(Z′Z)⁻¹Z′X]⁻¹ X′Z(Z′Z)⁻¹Z′u

    取概率极限：

    plim β̂_2SLS = β + plim{[X′Z(Z′Z)⁻¹Z′X]⁻¹ X′Z(Z′Z)⁻¹Z′u}

    根据大数定律：

    n⁻¹Z′Z →_p Q_ZZ（正定矩阵）

    n⁻¹Z′X →_p Q_ZX（满列秩，即相关性条件）

    n⁻¹Z′u →_p E(Z′u) = 0（外生性条件：Cov(Z,u)=0）

    因此：

    plim β̂_2SLS = β + (Q_XZ Q_ZZ⁻¹ Q_ZX)⁻¹ Q_XZ Q_ZZ⁻¹ · 0

    = β + 0 = β

    证毕。

    📌 考试推导技巧：

    ① 一致性证明的第一步永远是将y = Xβ + u代入估计量公式

    ② 第二步：分解出β + 额外的项

    ③ 第三步：对额外的项取plim

    ④ 第四步：用大数定律展开每一项

    ⑤ 第五步：利用Cov(Z,u)=0消去偏误项

    这个套路适用于所有估计量的一致性证明（OLS、IV、2SLS、GMM等）！
